# Movie Recommender

A fast and intuitive web application to discover movies based on your preferences.

## Features

- **Genre Filtering** - Browse movies by category (Action, Romance, Comedy, Horror, Drama, Fantasy)
- **Rating Filter** - Find highly-rated movies (7+, 8+, 9+ ratings)
- **Search Functionality** - Search for movies with auto-suggestions
- **User Authentication** - Sign in to manage your favorite movies
- **Favorites List** - Save movies you love (requires sign in)
- **Movie Details** - Click on any movie card to see more information
- **Responsive Design** - Works seamlessly on desktop and mobile devices

## Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No installation or server setup required

### Installation

1. Clone or download this project
2. Open `index.html` in your web browser
3. Start discovering movies!

## How to Use

1. **Browse by Genre** - Select a genre from the dropdown menu
2. **Set Minimum Rating** - Choose your preferred minimum rating
3. **Click Recommend** - Get personalized movie recommendations
4. **Search Movies** - Use the search bar in the navbar to find specific movies
5. **Save Favorites** - Sign in and click the heart button to add movies to your favorites

## Technologies Used

- **HTML5** - Page structure
- **CSS3** - Styling and animations
- **JavaScript (Vanilla)** - Dynamic functionality
- **JSON** - Movie database
- **Local Storage** - User preferences and favorites

## Features in Detail

### Movie Recommendations
Filter movies by genre and minimum rating to get personalized recommendations instantly.

### Search & Suggestions
Real-time search functionality with auto-suggestions for quick access to specific movies.

### User Accounts
Create accounts and manage your favorite movies. Your preferences are saved in your browser's local storage.

### My List
View all your favorite movies in one place by clicking "My List" in the profile menu.
